# Система аналитики

- `data_to_dasboards.ipynb` - собирает на основе основной витрины данных аналитическую витрину, необходимую для построения дашбордов. \
Скрипт использует только библиотеки pandas и numpy, на вход принимает выгрузки данных из основной витрины. Пример во вложении `nerd_analytics_v19.xlsx`
- `Система аналитики.docx` - требования к системе дашбордов
