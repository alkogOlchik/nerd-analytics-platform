# Гайд: показать пошаговую инструкцию, как совершить покупку на сайте
Сайт: http://localhost:8099/freshmart/

## Шаг 1
![скриншот](../screenshots/step_001.jpg)
[Блок ?] Откройте браузер и перейдите по указанному адресу для начала работы.

## Шаг 2
![скриншот](../screenshots/step_002.jpg)
[Блок 1] Выберите один из продуктов, например, «Яблоки», чтобы продолжить покупку.

## Шаг 3
![скриншот](../screenshots/step_003.jpg)
[Блок 3] Заполните информацию о доставке и нажмите «Подтвердить заказ» для оформления покупки.

## Шаг 4
![скриншот](../screenshots/step_004.jpg)
[Блок 4] Введите поисковый запрос в поле и выполните поиск товаров на сайте.

## Шаг 5
![скриншот](../screenshots/step_005.jpg)
[Блок 5] Выберите опцию «Инструменты» для продолжения совершения покупки на сайте.
