# Справочник продуктов — Нёрд-аналитика

Нёрд-аналитика — платформа поддержки клиентов. К ней подключены следующие продукты компании:

| Название продукта | Тип | Описание |
|-------------------|-----|----------|
| **NerdShop** | веб-сервис | Онлайн-магазин: каталог, заказы, доставка |
| **NerdPay** | платёжный сервис | Платежи, подписки, тарифы, счета-фактуры |
| **NerdGo** | мобильное приложение | Мобильное приложение для заказа услуг и доставки |
| **NerdAPI** | API интеграция | REST API-платформа для разработчиков |
| **NerdDesk** | личный кабинет | Единый портал самообслуживания клиента |
| **NerdBI** | аналитический модуль | Бизнес-аналитика, дашборды, отчёты |

## Синонимы и сопоставление

Пользователи могут называть продукты по-разному. Вот как они соответствуют:

- «веб-сервис», «сайт», «онлайн-магазин», «магазин» → **NerdShop**
- «платёжный сервис», «оплата», «платежи», «NerdPay» → **NerdPay**
- «мобильное приложение», «приложение», «NerdGo», «мобилка» → **NerdGo**
- «API интеграция», «API», «NerdAPI», «разработка», «интеграция» → **NerdAPI**
- «личный кабинет», «ЛК», «кабинет», «аккаунт», «NerdDesk» → **NerdDesk**
- «аналитический модуль», «аналитика», «дашборд», «NerdBI» → **NerdBI**

## Общая платформа

Все продукты используют единую платформу поддержки **Нёрд-аналитика**:
- Пользователь описывает проблему в чате → ИИ-помощник отвечает по базе знаний нужного продукта
- Если ИИ не справился → обращение передаётся оператору и превращается в тикет
- Пользователь отслеживает статус тикета в личном кабинете NerdDesk
- Операторы работают с очередью тикетов в административной панели

## Категории обращений (одинаковые для всех продуктов)

| Категория | Описание |
|-----------|----------|
| Технический сбой | Сервис недоступен, ошибки 500, критические неисправности |
| Ошибка в данных | Неверные числа, дублирование, расхождение отчётов |
| Проблема с доступом | Не войти в аккаунт, заблокирован, потерян пароль |
| Вопрос по оплате | Вопросы о тарифе, списаниях, счетах |
| Запрос возврата | Ошибочный платёж, двойное списание, возврат средств |
| Жалоба на сервис | Претензии к качеству обслуживания или работе операторов |
| Запрос документов | Акты, счета-фактуры, справки, выгрузки |
| Консультация | Вопросы о функционале, как что-то сделать |
