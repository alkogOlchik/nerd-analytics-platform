# NerdBI — аналитическая платформа

**Тип продукта:** аналитический модуль  
**Описание:** NerdBI — платформа бизнес-аналитики с дашбордами, графиками и отчётами. Предоставляет данные по обращениям, пользователям, эффективности поддержки и отзывам в режиме реального времени.

**Поддержка через платформу:** Нёрд-аналитика  
**Доступ:** только для сотрудников компании (роли: Аналитик, Администратор, Владелец продукта)

## Часто задаваемые вопросы — NerdBI

### Данные в NerdBI не совпадают с выгрузкой
Возможные причины:
1. **Задержка обновления** — данные в NerdBI обновляются раз в 1 час. Подождите и обновите страницу.
2. **Разные фильтры** — убедитесь, что на дашборде и в выгрузке выбраны одинаковые период и продукт.
3. **Часовой пояс** — проверьте настройки часового пояса в профиле. Выгрузка CSV может использовать UTC.
4. Если расхождение системное — создайте обращение «Ошибка в данных» с конкретными числами и периодом.

### Дублируются записи в таблице NerdBI
1. Проверьте, не применены ли пересекающиеся фильтры.
2. Убедитесь, что не выбрано несколько одинаковых периодов.
3. Если дублирование сохраняется после сброса фильтров — создайте тикет «Ошибка в данных» с примером записей (ID, дата, значения).

### Дашборд не загружается / пустые графики
1. Обновите страницу (F5).
2. Очистите кэш браузера.
3. Попробуйте уменьшить период: большой диапазон (более 1 года) может вызывать timeout.
4. Попробуйте экспортировать данные в CSV, если визуализация не работает.
5. Если ничего не помогло — создайте тикет «Технический сбой».

### Как экспортировать данные из NerdBI?
На каждом графике есть кнопка экспорта (иконка загрузки справа от заголовка):
- **PNG** — снимок графика для презентаций
- **PDF** — весь дашборд как отчёт
- **CSV** — сырые данные под графиком для анализа в Excel

### Нужна выгрузка данных за квартал / кастомный отчёт
1. Выберите нужный период через date picker на дашборде.
2. Примените нужные фильтры (продукт, категория, приоритет).
3. Нажмите кнопку экспорта CSV/PDF.
4. Для нестандартного формата или большого объёма данных — создайте обращение «Запрос документов».

### Как найти аномалии в данных?
- Дашборд «Тикеты» → карточка «Текущие аномалии» — автодетекция всплесков за последние 48 часов.
- Спросите ИИ-ассистента: «Найди аномалии за последние 7 дней» или «Что выросло за эту неделю?».
- Дашборд «Сводная информация» → график «Динамика обращений» — визуальный поиск выбросов.

### Как получить доступ к NerdBI?
Доступ предоставляется администратором компании. Обратитесь к своему руководителю или владельцу продукта — он пригласит вас по email с назначением роли (Аналитик / Оператор / Администратор).

## Дашборды NerdBI

| Дашборд | Что показывает |
|---------|----------------|
| Сводная информация | Общее число обращений, открытые тикеты, возвраты, удовлетворённость |
| Эффективность ИИ | % проблем решённых ИИ, топ эскалированных категорий, сравнение ИИ vs человек |
| Эффективность операторов | Нагрузка, TTFR, TTR, соблюдение SLA, рейтинг операторов |
| Пользователи | Число клиентов, демография, retention, активность |
| Отзывы | Тональность, топ ключевых слов, динамика позитива/негатива |
| Тикеты | Категории, аномалии, тепловая карта по времени, прогноз |

## Типичные проблемы и решения — NerdBI

| Проблема | Категория | Быстрое решение |
|----------|-----------|-----------------|
| Данные не совпадают с выгрузкой | Ошибка в данных | Проверить фильтры и часовой пояс |
| Дублирование записей | Ошибка в данных | Сбросить фильтры, затем тикет |
| Дашборд не загружается | Технический сбой | Уменьшить период, очистить кэш |
| Нет доступа к NerdBI | Проблема с доступом | Запросить у администратора компании |
| Нужна кастомная выгрузка | Запрос документов | Тикет с описанием формата |
| Как читать конкретный график | Консультация | ИИ-ассистент: «Объясни этот график» |
