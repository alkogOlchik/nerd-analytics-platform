# NerdDesk — личный кабинет

**Тип продукта:** личный кабинет  
**Описание:** NerdDesk — единый портал самообслуживания для клиентов компании. Здесь можно управлять профилем, просматривать историю заказов и платежей, скачивать документы, менять настройки подписки и обращаться в поддержку.

**Поддержка через платформу:** Нёрд-аналитика

---

## Часто задаваемые вопросы — NerdDesk

### Не могу войти в NerdDesk
1. Нажмите «Забыли пароль?» на странице входа → введите email регистрации.
2. Проверьте, каким способом регистрировались: email/пароль, Google, GitHub или соцсети.
3. Если аккаунт заблокирован после 5 неудачных попыток — подождите 30 минут.
4. Не приходит письмо восстановления? Проверьте папку «Спам» и «Промоакции».
5. Если ничего не помогло — создайте обращение «Проблема с доступом» (можно без входа в кабинет).

### Как изменить email или пароль?
NerdDesk → Настройки → Безопасность:
- Смена email: введите новый → придёт письмо с подтверждением на оба адреса.
- Смена пароля: введите текущий → введите новый дважды → сохранить.

### Где найти историю заказов / платежей?
NerdDesk → «Мои заказы» (для заказов NerdShop / NerdGo) или «Платежи» (для оплат через NerdPay). Есть фильтрация по дате и статусу.

### Как скачать закрывающие документы?
NerdDesk → «Документы». Документы формируются автоматически:
- Акты выполненных работ: до 5-го числа следующего месяца
- Счета-фактуры: одновременно с актами
- Чеки: сразу после оплаты

Если нужного документа нет — создайте обращение «Запрос документов».

### Медленно загружается NerdDesk / страница с ошибкой
1. Обновите страницу (F5).
2. Очистите кэш браузера (Ctrl+Shift+Delete → «Изображения и файлы», «Файлы cookie»).
3. Попробуйте другой браузер или режим инкогнито.
4. Проверьте интернет-соединение.

### Как удалить аккаунт NerdDesk?
NerdDesk → Настройки → Аккаунт → «Удалить аккаунт». Это действие необратимо. После запроса на удаление:
- Данные сохраняются 30 дней (период ожидания по GDPR).
- В течение этого времени можно отменить удаление.
- После 30 дней все данные удаляются безвозвратно.

### Жалоба на качество обслуживания
Создайте обращение «Жалоба на сервис» с подробным описанием: дата, суть претензии, имя сотрудника (если известно). Официальные жалобы рассматриваются в течение 5 рабочих дней.

## Типичные проблемы и решения — NerdDesk

| Проблема | Категория | Быстрое решение |
|----------|-----------|-----------------|
| Не войти, забыл пароль | Проблема с доступом | «Забыли пароль?» → email |
| Кабинет не загружается | Технический сбой | Очистить кэш, сменить браузер |
| Нет нужного документа | Запрос документов | ЛК → Документы или создать тикет |
| Неверный баланс/история | Ошибка в данных | Тикет с конкретными расхождениями |
| Претензия к оператору | Жалоба на сервис | Тикет с описанием ситуации |
| Как настроить уведомления | Консультация | Настройки → Уведомления |
