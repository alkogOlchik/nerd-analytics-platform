# Демо-сайты продуктов — адреса и описания

Платформа Нёрд-аналитика интегрирована со следующими продуктами. У каждого продукта есть свой сайт/приложение.

## Адреса сайтов (локальный стенд, порт 8099)

| Продукт | Сайт | URL |
|---------|------|-----|
| NerdShop | FreshMart — онлайн-магазин продуктов | http://localhost:8099/freshmart/ |
| NerdPay | FastPay — сервис быстрых платежей | http://localhost:8099/fastpay/ |
| NerdGo | GoExpress — сервис доставки | http://localhost:8099/goexpress/ |
| NerdAPI | DevBridge — API-платформа для разработчиков | http://localhost:8099/devbridge/ |
| NerdDesk | MyProfile — личный кабинет пользователя | http://localhost:8099/myprofile/ |
| NerdBI | DataPulse — аналитический дашборд | http://localhost:8099/datapulse/ |

## FreshMart (NerdShop) — http://localhost:8099/freshmart/

Онлайн-магазин свежих продуктов. Сайт позволяет:
- Просматривать каталог товаров (фрукты, овощи, деликатесы)
- Открывать карточку товара с описанием и выбором количества
- Добавлять товары в корзину
- Оформлять заказ с выбором даты и времени доставки
- Выбирать способ оплаты (карта онлайн, СБП, при получении)

Навигация: Каталог → Корзина → Оформление → Подтверждение заказа

## FastPay (NerdPay) — http://localhost:8099/fastpay/

Сервис быстрых денежных переводов. Сайт позволяет:
- Просматривать баланс и последние операции на главной странице
- Переводить деньги по номеру телефона или карты
- Выбирать получателей из списка быстрого доступа
- Подтверждать перевод перед отправкой
- Просматривать историю операций с фильтрами

Навигация: Главная (баланс) → Перевод → Подтверждение → Успех

## GoExpress (NerdGo) — http://localhost:8099/goexpress/

Сервис курьерской доставки. Сайт позволяет:
- Оформить заказ на доставку: указать адрес откуда и куда
- Выбрать размер посылки (Конверт, Коробка S, Коробка L) с расчётом стоимости
- Отследить статус заказа по номеру с временной шкалой и информацией о курьере

Навигация: Заказать доставку → Форма заказа → Успех → Отследить заказ

## DevBridge (NerdAPI) — http://localhost:8099/devbridge/

API-платформа для разработчиков. Сайт позволяет:
- Изучить документацию API (введение, авторизация, webhooks, коды ошибок)
- Создать и управлять API-ключами (test/live окружения)
- Тестировать API-запросы в интерактивном Playground

Навигация: Обзор → Документация → API Ключи → Playground

## MyProfile (NerdDesk) — http://localhost:8099/myprofile/

Личный кабинет пользователя. Сайт позволяет:
- Просматривать профиль: личные данные, контакты, подключённые сервисы
- Редактировать личные данные (имя, email, телефон, город)
- Управлять безопасностью: пароль, 2FA, активные сессии

Навигация: Обзор → Редактировать → Безопасность

## DataPulse (NerdBI) — http://localhost:8099/datapulse/

Аналитический дашборд. Сайт позволяет:
- Смотреть ключевые метрики: пользователи, сессии, конверсия, отказы
- Анализировать активность пользователей на графиках
- Просматривать топ страниц по посещаемости
- Управлять источниками данных (PostgreSQL, Kafka, REST API, Webhook)
- Скачивать готовые аналитические отчёты

Навигация: Дашборд → Источники → Отчёты
