# Нёрд-аналитика — платформа поддержки клиентов

Нёрд-аналитика — это единая платформа поддержки, к которой подключены все продукты компании.

## Продукты компании на платформе

| Продукт | Тип | Описание |
|---------|-----|----------|
| **NerdShop** | веб-сервис | Онлайн-магазин: каталог товаров, заказы, доставка |
| **NerdPay** | платёжный сервис | Платежи, подписки, тарифы, закрывающие документы |
| **NerdGo** | мобильное приложение | Мобильное приложение для заказа услуг и доставки |
| **NerdAPI** | API интеграция | REST API-платформа для разработчиков |
| **NerdDesk** | личный кабинет | Единый портал самообслуживания клиента |
| **NerdBI** | аналитический модуль | Бизнес-аналитика, дашборды и отчёты |

## Как работает поддержка

Пользователь описывает проблему в чате → ИИ-помощник ищет ответ в базе знаний нужного продукта → если ИИ не справился, обращение передаётся живому оператору → создаётся тикет → пользователь отслеживает статус в NerdDesk.

## Что такое Нёрд-аналитика

Нёрд-аналитика — платформа-посредник между пользователями и продуктами компании. Она:
- агрегирует все обращения в едином окне
- автоматически классифицирует и приоритизирует тикеты
- предоставляет аналитику для менеджеров (дашборд NerdBI)
- снижает нагрузку на операторов через ИИ-помощника первой линии

Нёрд-аналитика — это **платформа**, а NerdShop, NerdPay, NerdGo и другие — это **продукты**, которые к ней подключены.
