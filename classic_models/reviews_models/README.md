# Reviews Models

В этой папке лежат классические модели для обработки отзывов. Каждая подпапка отвечает за отдельную задачу и содержит ноутбуки обучения, скрипт инференса и подробный README.

## `categories`

Модель определения категории отзыва (`final_category`).

В папке лежат:

- ноутбуки для обучения и сравнения вариантов модели;
- ноутбук для сохранения препроцессоров;
- `predict_categories.py` для запуска предсказаний;
- подробный `README.md` по задаче категорий.

## `ratings`

Модель предсказания оценки отзыва (`rating`).

В папке лежат:

- ноутбуки для обучения и сравнения вариантов модели;
- ноутбук для сохранения препроцессоров;
- `predict_ratings.py` для запуска предсказаний;
- подробный `README.md` по задаче рейтинга.

## `sentiments`

Модель определения тональности отзыва (`sentiment`).

В папке лежат:

- ноутбуки для обучения и сравнения вариантов модели;
- ноутбук для сохранения препроцессоров;
- `predict_sentiments.py` для запуска предсказаний;
- подробный `README.md` по задаче тональности.
